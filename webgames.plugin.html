A simple implementation of a maze game using DFS to build the maze. I mostly focused on a small engine I'm writing in TypeScript that you can find in the repo for this project. The main impetus was to give ECS a try and see how it goes. The implementation is okay but needs some work.

My first TypeScript game. This project was just to familiarize myself with the language. You play as X against a minimax implementation. It should be impossible for you to win unless I made a mistake. (I don't think I did.)
